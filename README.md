# MultiTypeAdapter
RecyclerView adapter for support multi types items 

Gradle
------
Add it in your root build.gradle at the end of repositories:
```
allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```

Add the dependency
```
dependencies {
	        compile 'com.github.alexklab:MultiTypeAdapter:1.0.2'
	}
```
